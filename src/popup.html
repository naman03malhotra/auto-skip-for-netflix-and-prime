Auto Skip for Prime and Netflix (ASPN)

You have done all the hard work, sit back relax and binge-watch. This extension starts working as soon as you install it (refresh once) and works both for Prime and Netflix. What does it do for you?

  • Now it supports all 22 languages on prime, except skip Ads, which only works for a few languages.
  • To enable skip Ads in your language please add it here
  • Auto Skips intros and recaps on both Prime and Netflix
  • Auto Skips credits and play next episode on Prime
  • Auto Skips Ads on Prime (Not IMDB Ads)
  • Auto Skips "Continue watching on Netflix"

It is not working for me:

  • Please refresh the Netflix and Prime page once after you install it.
  • If it is still not working for you, please switch your language to English. It should work smoothly for Netflix and Prime.
  • Raise an issue here, preferably a PR. If you want to make it work in your local language.
  • Reason it might not be working for you is one of the translation is not correct, check here.
  • You can also drop an email if you are not familiar with github.
